# Space Tourism

This is a frontend practice project using the assets and designs provided [here](https://www.frontendmentor.io/challenges/space-tourism-multipage-website-gRWj1URZ3).

I aim to try out the following technologies

- Next.js v12
- Tailwind v3
- Aviv/WebP image formats
- Use a css grid structure
- Auto deployment to Vercel

### Installation

```
git clone git@github.com:puyanwei/space-tourism.git
npm install
npm run dev
```

The page will be live at this [url](space-tourism-murex.vercel.app).
